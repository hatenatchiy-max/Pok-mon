# ポケモンチャンピオンズ 相性補完チェッカー

ポケモンチャンピオンズの確認済みポケモンから最大6匹を選び、タイプ相性補完を確認できる静的Webアプリです。既存のメガシンカ形態は別カードとして追加しています。

## 使い方

`index.html` を直接開くか、ワークスペース直下で次のようにローカルサーバーを起動して開きます。

```sh
python3 -m http.server 4173
```

ブラウザでは次のURLを開きます。

```text
http://127.0.0.1:4173/champions-type-checker/
```

## チェック内容

- 攻撃の相性: 選択中ポケモンのタイプ一致技だけを想定し、単タイプ18種と内定ポケモン全体への最高打点を表示
- 防御の相性: 各攻撃タイプに対して、弱点になる味方、耐性・無効で受けられる味方を表示
- 全体の相性補完: 攻撃面と防御面を50%ずつで簡易スコア化し、追加候補を表示

## メガシンカの扱い

メガシンカ形態は通常形態とは別カードです。通常形態を選択中に同じポケモンのメガシンカ形態をクリックすると、チーム内では通常形態からメガシンカ形態へ置き換わります。逆にメガシンカ形態を選択中に通常形態をクリックした場合も置き換わります。

メガシンカの評価ではタイプ変化のみを反映します。特性、技範囲、種族値、持ち物などは評価対象外です。

## データ前提

内定ポケモン一覧は2026-06-15時点で確認したMeriStationの記事をベースにしています。記事内で同名重複しているポケモンは、ライチュウ（アローラ）、キュウコン（アローラ）、ウインディ（ヒスイ）、ヤドラン（ガラル）、ジュナイパー（ヒスイ）として整理しています。また、記事上で連結されていた `GoodraKlefki` はヌメルゴンとクレッフィに分割しています。メガシンカは、このリスト内の通常ポケモンが持つ既存メガシンカ形態を追加しています。

タイプ判定は第9世代相当の標準タイプ相性のみを使います。特性、技範囲、持ち物、テラスタル、Zワザ、種族値、役割は評価対象外です。
